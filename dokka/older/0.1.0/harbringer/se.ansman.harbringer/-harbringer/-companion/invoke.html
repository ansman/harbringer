Creates a new Harbringer with the given storage, maxRequests, and maxDiskUsage.

Parameters

storage

The storage to use for the requests.

maxRequests

The maximum number of requests to store. If this is exceeded, the oldest requests will be deleted.

maxDiskUsage

The approximate max size of the storage in bytes. If this is exceeded, the oldest requests will be deleted. In-flight requests are not counted towards this limit, so more data can be stored temporarily.

maxAge

The maximum age of the requests to store. If this is exceeded, the oldest requests will be deleted.

enabled

If true (default), requests will be stored. If false, requests will not be stored.

scrubber

A lambda that returns a Scrubber for the given Request. This can be used to redact sensitive data.
