Harbringer

interface Harbringer

A logger for HTTP requests and responses.

This logger can be used to log requests and responses for debugging purposes, or to store them for later analysis.

The requests are stored in a se.ansman.harbringer.storage.HarbringerStorage and can be persisted across sessions.

You can export the logged requests to a HAR archive using exportHarArchive, which can then be imported into tool for analysis.

This class is thread safe and can be used from multiple threads. Unless otherwise noted, all methods and properties can perform I/O and should not be called from a UI thread.
