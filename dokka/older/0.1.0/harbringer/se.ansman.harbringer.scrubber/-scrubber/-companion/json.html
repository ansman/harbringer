fun json(vararg paths: String, json: <Error class: unknown class> = Json, onlyIf: (Harbringer.Request) -> Boolean = { true }, replace: (path: String, <Error class: unknown class>) -> <Error class: unknown class>? = defaultJsonReplacement): BodyScrubber

Returns a new BodyScrubber that scrubs a JSON body of sensitive keys.

The paths list is a list of JSON paths to scrub.

An example of a path looks like this: $.array[].key

By default, matching objects will be replaced with an empty object, arrays with an empty array, and primitives with "******". Null values will be kept as null

The body will be replaced if the onlyIf lambda returns true.

Parameters

paths

A list of paths to scrub. The keys are case-sensitive.

json

The Json instance to use for parsing and encoding the JSON. Defaults to Json.

onlyIf

A lambda that takes a Harbringer.Request and returns true (default) if the body should be replaced.

replace

A lambda that takes the path and the element to replace. If the lambda returns null then matches key is removed.
